Обобщение это переменный тип данных. Нужно это для того, 
если заранне не знаешь какой тип данных будет использоваться.
Обобщение позволяет указать тип данных непосредственно в момент вызова функции.
Пример: задача написать программу которая будет считывать определенный файл выбираемый пользователем.
Если файл заранее определен и лежит на диске то можно использовать явно типизированный метод типа string для получения пути к файлу.
Однако если заранне пользователь не знает будет этот файл на диске с путем типа string либо с базы данных имеющий целочисленный номер в виде пути к файлу
то тут поможет обобщение.
